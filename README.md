# Análisis de Ventas – Online Retail

## Contexto de Negocio
Una empresa de comercio electrónico busca comprender mejor su desempeño en ventas y el comportamiento de compra de sus clientes con el fin de identificar patrones que permitan tomar decisiones basadas en datos.

## Objetivo
Analizar datos transaccionales de ventas para identificar tendencias, patrones de comportamiento de los clientes y generar insights accionables que apoyen la optimización del negocio.

## Descripción del Conjunto de Datos
El conjunto de datos contiene información histórica de transacciones, incluyendo:
- Identificador de factura
- Detalles de productos
- Cantidad y precio unitario
- Identificador de cliente
- Fecha de la transacción

> Nota: El conjunto de datos original no se incluye en este repositorio.

## Metodología
- Revisión y comprensión de los datos
- Limpieza y preparación del dataset
- Análisis exploratorio de datos (EDA)
- Análisis descriptivo para identificar patrones relevantes

## Principales Hallazgos
- Identificación de productos con mayor volumen de ventas
- Análisis de comportamiento de compra de los clientes
- Tendencias generales en las transacciones

## Recomendaciones de Negocio
- Enfocar estrategias comerciales en clientes recurrentes
- Priorizar productos con mejor desempeño
- Utilizar los patrones identificados para apoyar la toma de decisiones comerciales

## Herramientas Utilizadas
- Python
- Pandas
- NumPy
